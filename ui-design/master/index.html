Classifications

Enterprise application software

Business applications used to satisfy the needs of an organization.

Resource Management

Applications that facilitate the efficient and effective deployment of an organization's resources.

Dashboard Applications

Applications that are capable of displaying summaries of business data that show at-a-glance understanding of conditions through metrics and key performance indicators.

Online analytical processing

Interactive examination of large amounts of data from many perspectives.

Reporting Application

Generates aggregated views of data to keep the management informed about the state of their business.

Procurement Application

Automate the purchasing function of organizations.

Data mining

Takes charge of the extraction of consumer information from a database by utilizing software that can isolate and identify previously unknown patterns or trends in large amounts of data.

Business performance management

Applicationn with management and analytical processesing modules that enables the management of an organization's performance to achieve one or more pre-selected goals.

Document management

Made for organizing and management of multiple business documents of various types.

Used for creating and distributing employee schedules, as well as tracking employee hours.

Introduction to Mobile Application Development

As smartphone and tablet sales continue to rise,
one thing is certain: Mobile computing is the future of business.

Mobile Apps come in two flavours

  • Native applications
  • Mobile web applications

The Difference

The biggest difference between Native and Web Mobile applications is determined by the framework used to develop the app. Native apps are written with native (compiled) code while Web apps are written using web technologies such as HTML 5.
